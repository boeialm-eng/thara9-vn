# CLAUDE.md — บริบทโปรเจกต์เกม RIASEC

## โปรเจกต์คืออะไร
ชุดเกม visual novel วัดความสนใจอาชีพ (RIASEC) สำหรับวัยรุ่นไทย 3 เกม:
- **thara9-vn.html** — ธีมอวกาศ (ต้นแบบหลัก)
- **cafe-game.html** — ธีมคาเฟ่
- **island-game.html** — ธีมเกาะ / เอาตัวรอด

ทั้ง 3 เกมมีบริบทเรื่องต่างกัน แต่วัดด้วยมาตรวัดเดียวกัน ผลลัพธ์จึงต้องเปรียบเทียบกันได้ข้ามเกม

---

## กฎเหล็ก (ห้ามละเมิด)

### แกนวัด — ห้ามแก้ตรรกะ
- `TYPE / DIMS / HEX / GROUP / analyze() / hexSVG()` ต้องเหมือน thara9-vn.html เป๊ะ
- ทุกจุดวิกฤตต้องมี 6 โดเมน R/I/A/S/E/C ครบสมดุล — **เขียน assert เช็กตอนโหลด ถ้าไม่ครบให้ console.error**
- คะแนนนับโดเมนล้วน ค่านิยม (dilemma) เก็บแยก ไม่ปนคะแนน
- `behaviorLog` เก็บเวลาเฉย ๆ ไม่คิดคะแนน

### ตัวเลือก — วินัยการวัด
- ตัวเลือก = การกระทำสด ๆ ไม่มีป้ายอาชีพ ไม่มีมุก ภาษากลางไม่ชี้นำ
- **สลับลำดับสุ่มทุกจุด**
- ห้ามทาง R/I ดูดีกว่าทางอื่นโดยโครงสร้าง

### ภาษา
- **ภาษาไทยล้วน** ห้ามคำอังกฤษปนในข้อความ ตัวเลือก ผลลัพธ์
- ศัพท์วิชาชีพมีวงเล็บอธิบาย เช่น "นักออกแบบ UX (ออกแบบประสบการณ์ผู้ใช้)"
- ห้ามตัวอักษรไทยในภาพที่เจน AI

### ภาษาผลลัพธ์
- ใช้คำว่า "แนวที่เธอโน้มไป / น่าสำรวจ / เปิดรับ" ห้ามใช้คำเหล่านี้:
  - "ตัวตนที่แท้จริง" / "คุณคือ..." / "เหมาะแน่นอน" / "ทำนาย" / "ถนัด"
- disclaimer ต้องชัดเจน: นี่คือแนวจากสิ่งที่เลือกทำในเรื่อง ไม่ใช่คำตัดสิน

### UI / เทคนิค
- ตัวละครทุกตัวเป็น `<img class="sprite" src="assets/...">` ไม่ใช่ SVG (SVG เป็น fallback ผ่าน onerror)
- หน้าผลลัพธ์เลื่อนได้สุด:
  ```css
  #result { align-items: flex-start; overflow-y: auto; -webkit-overflow-scrolling: touch; }
  #result .box { margin: 20px auto; max-height: none; }
  ```
- ปุ่มลดเอฟเฟกต์ + prefers-reduced-motion + กระพริบ ≤ 3 ครั้ง/วิ

### เก็บผลลัพธ์
ทุกเกมจบต้องเก็บข้อมูลลง localStorage **และ** console.log:
```js
const RESULT = { game: '<space|cafe|island>', ts: Date.now(), tally: {...tally}, code: r.code, values: valuesChoice, log: behaviorLog };
localStorage.setItem('thara_result_' + RESULT.game, JSON.stringify(RESULT));
// 🔌 จุดต่อระบบเก็บข้อมูลกลาง
```

## โครงสร้าง GROUP (มาตรฐาน — ต้องเหมือนกันทั้ง 3 เกม)
```js
const GROUP = {
  R: { name, desc, love, env, strength, grow, fac, jobs },
  I: { ... },
  A: { ... },
  S: { ... },
  E: { ... },
  C: { ... },
};
```
- **desc**: เป็นคนแบบไหน 3-4 ประโยค ภาษาไทยล้วน
- **love**: ชอบอะไร / สนุกกับอะไร
- **env**: เหมาะงานแบบไหน สภาพแวดล้อมที่พลังงานพุ่ง
- **strength**: จุดแข็ง (เฟรมบวก)
- **grow**: สิ่งที่ลองพัฒนาต่อ (เฟรมบวก ไม่ใช่ข้อเสีย)
- **fac**: คณะ ภาษาไทยล้วน (ไม่มีชื่ออังกฤษ)
- **jobs**: อาชีพจริง ภาษาไทยล้วน (ไม่มีชื่ออังกฤษ)
